# UAE Cycling Routes Platform 🚴‍♂️🇦🇪

Welcome to the **UAE Cycling Routes Platform**, a premium, high-performance web application designed to be the ultimate guide for cyclists in the United Arab Emirates.

## 🌟 Features

- **Interactive Maps**: Real-time Leaflet-based maps showing track locations across all emirates.
- **Track Database**: Detailed information on Desert, Urban, and Mountain tracks (Length, Difficulty, Features).
- **Live Weather Integration**: Real-time temperature and wind speed for every track.
- **Smart Tools**: Advanced calculators and planners for a safe and efficient ride.
- **360° Virtual Tours**: HD tours to explore track terrains before you visit.
- **Premium Branding**: Official UAE Team Emirates 2026 gear and aesthetics.
- **PWA Ready**: Mobile-first design that can be installed on your device.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Mapping**: [Leaflet.js](https://leafletjs.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Data**: Open-Meteo API for weather.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/uae-cycling-website-project.git
   ```

2. Navigate to the project directory:
   ```bash
   cd uae-cycling-website-project
   ```

3. Install dependencies:
   ```bash
   npm install
   # or
   bun install
   ```

4. Run the development server:
   ```bash
   npm run dev
   # or
   bun dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `src/app`: Main pages and layouts.
- `src/components`: UI components (Map, Weather, Smart Tools).
- `public/images`: High-quality assets and track visuals.
- `public/tours`: Virtual tour data.
